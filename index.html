SAE 101

Hygiène Informatique et Cybersécurité

Projet centré sur la sensibilisation à l'hygiène informatique et à la cybersécurité. Étude des bonnes pratiques pour protéger les données personnelles et professionnelles, familiarisation avec les principaux risques numériques (phishing, malwares, mots de passe faibles, etc.).

Cybersécurité Sensibilisation Protection données
SAE 102

Initiation aux Réseaux Informatiques

Configuration d'une Raspberry Pi pour administration à distance via SSH. Commande de LED, interfaçage de capteurs température/humidité, et transmission de données sur serveur local/distant. Livrables : dossier technique, compte rendu et présentation.

Raspberry Pi SSH IoT
SAE 103

Dispositif de Transmission

Développement de compétences en analyse et interprétation de documents techniques liés aux mesures et transmissions. Compréhension des documents de mesure, analyse des schémas et câblages, caractérisation des supports de transmission (câbles, fibres optiques).

Transmission Fibre optique Câblage
SAE 104

Présence sur Internet - Ce Portfolio

Création d'un portfolio en ligne pour se présenter professionnellement sur internet. Pages de présentation, CV en ligne, galerie de projets et formulaire de contact. Utilisation de HTML, CSS et JavaScript pour une identité numérique soignée avec design attractif et fonctionnel.

HTML/CSS JavaScript Design Web
SAE 105

Traitement de Données

Projet axé sur le traitement des données : création de comptes utilisateurs, politique de mots de passe solides, commandes GNU/Linux. Chiffrement des données avec création d'une partition chiffrée sous Debian via LUKS pour garantir sécurité et confidentialité dans des environnements sensibles.

Linux Chiffrement LUKS
